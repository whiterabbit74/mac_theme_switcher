# ThemeSwitcher v1.0

## Quick Start

1. Double-click ThemeSwitcher to run
2. Look for the icon in your menu bar (top right)
3. Left click = Toggle theme
4. Right click = Settings menu

## Requirements

- macOS 12.0 or later
- First run requires AppleScript permissions (will be prompted)

## Permissions

When you first run ThemeSwitcher, macOS will ask for permission to control "System Events". This is required to change the system theme.

Click "OK" to grant access, or:
- Go to System Preferences ‚Üí Security & Privacy ‚Üí Privacy ‚Üí Automation
- Enable "System Events" for ThemeSwitcher
