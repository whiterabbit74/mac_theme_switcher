# ThemeSwitcher v1.1 - With Autostart

## NEW in v1.1
✨ Autostart functionality!
- Toggle autostart in settings menu
- Checkbox in settings window
- Automatic launch at login

## Installation

1. Download and extract ThemeSwitcher-v1.1-macOS.zip
2. Drag ThemeSwitcher.app to your Applications folder (optional)
3. Double-click ThemeSwitcher.app to run
4. Grant AppleScript permissions when prompted

## Usage

- **Left click menu bar icon**: Toggle theme instantly
- **Right click menu bar icon**: Settings menu with autostart option
- **Settings window**: Access via right-click menu for full configuration

## Autostart Setup

Two ways to enable autostart:
1. **Via menu**: Right-click the icon → 🚀 Autostart
2. **Via settings**: Right-click → ⚙️ Settings → Check autostart box

## Requirements

- macOS 12.0 or later
- AppleScript permissions (will be prompted)
